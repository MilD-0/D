Introduction

There are many great games that can't be fully experienced becasue of language barriers, although some of them get picked up by localisation companies, most of them are never translated except when dedicated fans pick up the task.

Many tools exist that automatically translate a game but they were never quite satisfying, though by implementing the latest A.I LLMs the translation are more accurate and readable than ever especially in the JP to ENG scene, tools like translator++ which this guide will focus on.

What you need

You need the following to Start transltaing:

  • The latest Version of Translator++.
  • A japanese game you want to translate.
  • A good internet connection or an offline translation engine.
Translator++

Translator++ is a CAT (Computer Aided Translation) or computer-aided human translation (CAHT) software for games. It helps translators to produce quality translation efficiently and quickly for various types of game engines such as RPG Makers, Wolf RPG Editor, RenPy, KiriKiri, and many more.

This tool is fully in GUI and equipped with convenient tools crafted to improve your translation experience.

Whether you are full time professional translator or just a hobbyist, you definitely will get the benefit from this tool.

Installing
Installation of Translator++ is pretty straightforward and easy, justdownload the archive from this link (and make sure to have winrar) then extract it and run Translator++.exe to start the application.
Starting

After running the program, select start new project, then select the engine the which the game runs on then select the game itself.

Then click create project, after which you must wait while the tool is extracting the text so it will be ready for translation.

Translating

After Translator++ has created the workspace for your game, go to option in the top right corner, choose your translation engine and language.

Next,go back to the main screen, where you will see a window listing the various text files the game has, you can click these files to translate individual elements or columns, but for simplicity just check the box right of the files on the window to perform a batch translation.

Now, with all the files checked click batch translation on the top of the screen, which will promt a window to show up, select your translation engine of choice then click translate now (this will take some time...) .

Applying Translation

To actually apply your translaion, click inject/apply translation on the top of the screen, where you will select the directory of the tranlated game, and choose a folder where a translated copy will be created .

Hit apply translation and wait.

Finally, your translated game should be in that directory for you to enjoy.

Limits

Some limitations which Translator++ may suffer from or some details this guide may not have touched:

  • This guide is meant for those who have no other option than to rely on AI translation and not for those looking to fully translate a game to share it, as it simplifies many elements of translating the game to only keep what you need to get the game from ja to en automatically and quickly.
  • Translator++ is a versatile tool, but it produces better results with RPG maker games and visual novels, while you need other steps to translate unity games, and it can only translate text.
  • some transaltion engines may not be able to translate some things while other ones will not be able to translate other thing, you should try 2 other if the first one doesn't do well.
  • of course, this is not a perfect tool, so some games will just not work, and proffesional human transaltion will always be better.
